# 🧾 Basic Informational Site with Node.js

This is a beginner-level Node.js project that builds a very basic informational website by manually serving static HTML files. The project demonstrates fundamental concepts in setting up an HTTP server, routing, and file handling using only core Node.js modules.

---

## 📚 Assignment Instructions

You will be creating a very basic informational site that contains 4 pages:

- `index.html` (Homepage)
- `about.html` (About page)
- `contact-me.html` (Contact page)
- `404.html` (Not Found page)

### URL Routing Requirements:

- `http://localhost:8080` should show `index.html`
- `http://localhost:8080/about` should show `about.html`
- `http://localhost:8080/contact-me` should show `contact-me.html`
- Any other URL should show `404.html`

---

## 📁 Project Structure

```
Basic-Informational-Site/
├── script.js       # Node.js server file
├── index.html     # Homepage
├── about.html     # About page
├── contact-me.html # Contact page
└── 404.html       # Not Found page
```
## 🔧 Setup & Run Instructions
### 1. **Ensure Node.js is installed Check by running this command in your terminal:**
```
node -v
```

### 2. **Navigate to the project directory Open your terminal and move into the project folder:**
```
cd path/to/Basic-Informational-Site
```

### 3. **Start the server Run the following command:**
```
node script.js
```

### 4. **Access the website Open your web browser and visit:**
```
http://localhost:8080
```
